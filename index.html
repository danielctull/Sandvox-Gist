Sandvox-Gist

Gist plug-in for Sandvox

This will allow you to easily add a gist to a Sandvox page.

Install

The easiest way is to download the prebuilt bundle and double click it. This will let Sandvox install the plug-in in the correct place.

Requrirements

Sandvox2 and a Mac running Mac OS X 10.6 (Snow Leopard) or later.

Download

The gist plugin can be downloaded here.

You can download the source code for this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/danielctull/Sandvox-Gist
